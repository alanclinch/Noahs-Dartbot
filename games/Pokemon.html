KEYBOARD SHORTCUTS

1–9, 0  → numbers 1–10 (0 = ten)
Q W E R T Y U I O P  → numbers 11–20
Shift+number → DOUBLE that number
Ctrl+number → TREBLE that number
(or click the on-screen DOUBLE/TREBLE buttons for sticky mode)
B  → Bull (25)  ·  Shift+B → Bullseye (D25)
M  → Miss
Backspace  → Undo last dart
Enter / Space  → Advance to next player
